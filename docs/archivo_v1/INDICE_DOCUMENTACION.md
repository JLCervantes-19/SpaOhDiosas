## 🎓 Glosario de Términos

- **Gestión de Citas**: Funcionalidad para cancelar/cambiar citas
- **Quick Replies**: Botones de respuesta rápida en el chat
- **24h de anticipación**: Requisito para cancelar/cambiar citas
- **Estado conversacional**: Punto actual en el flujo del chat
- **tempName**: Variable temporal para guardar nombre durante consulta
- **currentCitas**: Array de citas actuales del usuario
- **sessionId**: ID único de la sesión de chat
